Examples

Looping through a structure with a cell array of field names

student = webread('https://c-a-j.github.io/ENS1300/JSONexample.json'); 
fields = fieldnames(student.JoeSchmo); % creates cell array of field names

for i = 1:length(fields)
    field = fields{i};
    student.JoeSchmo.(field)
end

Looping through a structure with a string array of field names

student = webread('https://c-a-j.github.io/ENS1300/JSONexample.json'); 
fields = fieldnames(student.JoeSchmo); % creates cell array of field names
fields = string(fields); % converts cell array into string array

for i = 1:length(fields)
    field = fields(i);
    student.JoeSchmo.(field)
end
